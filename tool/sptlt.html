                SPTLT Operator Reference
                
            

            


                

                    
Route Editing

                    
    
                        
  • Drag Entry, Route nodes, HVA, Choke, and Exit Box to shape the transit path
    • 
                        
  • Click directly on the blue route line to insert a new waypoint at that position
    • 
                        
  • Right-click any numbered waypoint to delete it (minimum 4 waypoints enforced)
    • 
                        
  • Keep the route in water — use extra nodes to bend around islands and shoals
    • 
                        
  • Escorts may also be repositioned to refine the starting screen
    • 
                    

                


                

                    
Threat Geometry

                    
    
                        
  • Threat circles represent A2/AD range and influence areas — drag the red center dot to reposition
    • 
                        
  • FAC area is a polygon because littoral operating water is constrained by coast and islands
    • 
                        
  • Click a polygon edge to insert a new vertex; right-click a vertex handle to delete it
    • 
                        
  • Exit area stays a box — it represents a mission completion trigger zone
    • 
                    

                


                

                    
Coast / Shoal Guides

                    
    
                        
  • Coast Guide is a manual visual shoreline reference — adjust to match the satellite imagery
    • 
                        
  • Shoal Area marks shallow or restricted water that constrains the viable route corridor
    • 
                        
  • Click polyline edges or polygon edges to add vertices; right-click to remove
    • 
                        
  • Adjust both to help shape playable routes and danger zones before locking geometry
    • 
                    

                


                

                    
Save / Load / Reset

                    
    
                        
  • SAVE — downloads the current full state as mission_export.json
    • 
                        
  • LOAD — restores a previously saved JSON file back into the tool
    • 
                        
  • RESET — wipes all geometry back to the default BLACK TIDE Mission 1 layout
    • 
                    

                    
The exported JSON is the lock reference for translating layout into Sea Power .ini mission files.

                


                

                    
View Modes

                    
    
                        
  • DESIGN — full editing mode with sidebars, inspector, and all drag handles active
    • 
                        
  • BRIEFING — classified presentation view, hides editing UI, applies intel map filter
    • 
                    

                    
Use browser Print or screenshot in BRIEFING mode to export briefing images.

                


                

                    
Compilation Inspector

                    
    
                        
  • Right panel shows live Sea Power .ini output generated from current geometry
    • 
                        
  • INI Output covers unit positions and waypoint blocks — paste directly into mission files
    • 
                        
  • BriefingMap XML Helper covers threat circle overlays for briefing map XML
    • 
                        
  • Inspector updates automatically on every drag or geometry edit
    • 
                    

                


                

                    
Current Limits

                    
    
                        
  • Labels are fixed per zone — rename via JSON edit after export
    • 
                        
  • This tool does not auto-detect shoreline or bathymetry — operator judgment required
    • 
                        
  • Corridor offset may degrade on very sharp route bends — add intermediate waypoints to smooth
    • 
                        
  • Final .ini trigger logic must be authored manually after geometry is locked
    •
